Lesson 2: Thinking in HTML

Berners-Lee developed HyperText Markup Language (HTML), and in conjunction, he developed a protocol to access text from other documents via hyperlinks. The protocol was called HTTP, and this paved the way for the future. HTML itself was derived from a markup language called Standard Generalized Markup Language (SGML). Standardization being an ongoing process, modifications were constantly made and versions were released accordingly. The various versions of HTML that have been released are as follows:

✦ HTML 2.0 (November 1995)

✦3.2 (January 1997)

✦4.0 (December 1997)

✦4.01 (December 1999)

✦HTML5, which is widely used, but is still in the development stage
